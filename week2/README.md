# Week 2 – Programming Basics

## Overview
This week focuses on fundamental programming concepts including:
- Working with variables and swapping values without using a temporary variable
- Conditional logic (even/odd, positive/negative, leap year, etc.)
- Loops and operations on arrays and numbers
- Mathematical functions and calculations
- Printing numbers, multiplication tables, Fibonacci series, and computing LCM/GCD
