# Week 3 – String, Number, and Array Operations

## Overview
This week focuses on basic algorithms and operations involving:
- Strings: vowels counting, palindrome check, pattern printing
- Numbers: reversing, sum of digits, factorial, prime, Armstrong, Fibonacci
- Arrays: largest/smallest elements, sum, uniqueness, missing number
- Miscellaneous: BMI calculator, password strength, number guessing game, Collatz sequence, Pascal’s Triangle
