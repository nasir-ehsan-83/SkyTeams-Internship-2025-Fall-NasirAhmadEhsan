# Week 1 – Fundamentals, Testing, and GitHub Basics

## Overview
This week focuses on:
- Core programming concepts: variables, conditionals, loops, functions, arrays/lists
- Writing clean and modular code
- Using Git and GitHub for version control
- Writing and running basic unit tests

## Exercises

1. Swap two variables → `question-1.py`  
2. Check if a number is even, odd, or prime → `question-2.py`  
3. Sum all numbers in a list or array → `question-3.py`  
4. Reverse a string or list → `question-4.py`  
5. Find the largest number in a list → `question-5.py`  
6. Refactor messy code into readable functions → `question-6.py`  

## Git & GitHub Tasks
- Create repository: `Learning-Github`  
- Clone repository locally  
- Create branch for Week 1: `week-1`  
- Make frequent commits  
- Push changes to GitHub
