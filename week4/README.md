# Week 4 – Arrays, Strings, and Advanced Problem Solving

## Overview
This week focuses on:
- Arrays: second largest, merging, rotating, intersections, rearrangements
- Searching and sorting: binary search, kth largest element, top K frequent elements
- Strings: compression, uniqueness, rotation check, frequency, pattern matching, permutations
- Data structures: stack, queue
- Algorithms: Kadane’s, matrix multiplication, merge overlapping intervals
- Miscellaneous: ATM simulation, rock-paper-scissors, sentence abbreviation, balanced parentheses, palindromic substring, set matrix zeroes
